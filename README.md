Welcome to my personal portfolio website! This site showcases my journey in computer science, featuring my skills, projects, education, certifications, awards, and more. I'm Eric Wang (王书翰), a high school senior passionate about coding, math, and tennis. Fluent in English and Mandarin Chinese.

## Features

This portfolio includes a variety of interactive and informative sections:

- **Home**: Introduction, real-time clock, and contact info.
- **Skills**: Showcase of programming languages, frameworks, and tools I know.
- **Projects**: Extensive collection of my coding projects in Python, Java, C++, and web technologies, with images and GitHub links.
- **Certifications**: My achievements in IT and programming certifications.
- **Awards**: Academic and extracurricular awards.
- **Education**: My educational background.

## Technologies Used

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Backend**: Firebase (Authentication, Firestore, Realtime Database)
- **Frameworks/Libraries**: React (for Tetris game), Pygame (Python projects), etc.
- **Languages**: JavaScript, Python, Java, C++, HTML, CSS, PHP
- **Tools**: Git, GitHub, VS Code, Firebase CLI
- **Hosting**: GitHub Pages

## How to View

### Online
Simply visit [https://erichanwang.github.io](https://erichanwang.github.io) in your web browser.

### Locally
1. Clone the repository: `git clone https://github.com/erichanwang/erichanwang.github.io.git`

3. For full functionality (e.g., Firebase features), you may need to set up Firebase config (see wiki.md).
